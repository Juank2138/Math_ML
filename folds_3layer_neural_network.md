# Shallow neural networks and the number of folds

Consider a shallow neural network with a two-dimensional input
```math
\mathbf{x}=\begin{bmatrix}x_1\\ x_2 \end{bmatrix}
```
and three hidden units defined as
```math
h_1=a[\theta_{10}+\theta_{11}x_1+\theta_{12}x_2]
h_2=a[\theta_{20}+\theta_{21}x_1+\theta_{22}x_2]
h_3=a[\theta_{30}+\theta_{31}x_1+\theta_{32}x_2]
```
where each hidden neuron contains one weight for each input variable. The network output is obtained by combining the hidden units according to
```math
y=\phi_0+\phi_1h_1+\phi_2h_2+\phi_3h_3
```
The activation function is the Rectified Linear Unit (ReLU),
```math
a(z)=\max(0,z)=\begin{cases}0, & z<0,\\z, & z ≥ 0\end{cases}
```
Each hidden neuron changes its activation state when
```math
\theta_{i0}+\theta_{i1}x_1+\theta_{i2}x_2=0
```
which defines a hyperplane in the input space. These hyperplanes correspond to the folds of the network and partition the input space into linear regions.

The maximum number of regions generated by $H$ hyperplanes in a $D$-dimensional space is bounded by Zaslavsky's inequality,
```math
R(H,D)\leq \sum_{k=0}^{D}\binom{H}{k}
```
with equality when the hyperplanes are in general position. Therefore, a shallow ReLU network with $H$ hidden neurons can create at most
```math
\boxed{R(H,D)=\sum_{k=0}^{D}\binom{H}{k}
```
linear regions, where each additional hidden neuron introduces a new fold and increases the expressive power of the network.

\boxed{7 \text{ linear regions}}
