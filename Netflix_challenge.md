# Netflix Challenge Solution and its Relationship with Matrix Factorization

The 'Netflix Prize' and the associated 'KDD Cup 2007 Task 1' focused on improving recommendation systems by predicting user preferences for movies. Competitors were asked to estimate whether a user would rate a particular movie based on historical rating information.

One of the most successful approaches was presented in the *KDD Cup 2007 Task 1 Winner Report* by Miklós Kurucz and collaborators. Their solution combined several recommendation techniques, including matrix factorization methods and neighborhood-based models.
 
## Problem Description
The challenge consisted of predicting whether a user rated a movie during 2006 for a set of 100,000 user-movie pairs extracted from the Netflix Prize dataset.

The target variable was binary:
```math
y_{ij}=\begin{cases} 1 & \text{if user } i \text{ rated movie } j,\\ 0 & \text{otherwise}\end{cases}
```
Importantly, none of these user-movie pairs appeared in the training data, meaning that the task was not to predict the rating value itself, but rather the probability that an interaction occurred.

Predictions were evaluated using the 'Root Mean Squared Error' (RMSE) metric:
```math
RMSE=\sqrt{\frac{1}{N}\sum_{n=1}^{N}(y_n-\hat y_n)^2}
```
The winning solution combined several complementary recommendation approaches.

### 1. SVD-Based Recommendation
The first component relied on 'Singular Value Decomposition' (SVD) applied to the user-item interaction matrix. The matrix is approximated by a low-rank decomposition.

According to the 'Eckart-Young theorem', this decomposition provides the best rank-$k$ approximation of the original matrix with respect to the Frobenius norm. Since the Frobenius norm corresponds to the sum of squared prediction errors, minimizing it is closely related to minimizing RMSE.

### 2. Item-Item Collaborative Filtering
The second component was based on 'item similarity'. Instead of factorizing the entire matrix, this approach predicts the preference for a movie by looking at similar movies that the user has already rated. Movie similarity was computed using the 'adjusted cosine similarity'

Predictions for a target movie $j$ were generated using a weighted average of the $K$ nearest neighboring movies previously rated by the user. This approach captures local relationships between items that may not be represented by a global factorization model.

### 3. Sequential Association Rules
The third component exploited temporal information in user behavior. Movies rated by a user were ordered according to rating time, producing sequences such as
```math
\text{Movie A} \rightarrow \text{Movie B} \rightarrow \text{Movie C}
```
Frequent sequential patterns were extracted using the 'Apriori algorithm' to discover association rules of the form
```math
A,B \Rightarrow C
```
These rules identify movies that are frequently rated after observing previous viewing patterns. Because of computational limitations, the analysis was restricted to:

- movies with at most 50,000 ratings,
- users with at most 3,000 ratings.

As a result, this component produced predictions for approximately '20,000 of the 100,000' target pairs.

## Ensemble Learning
The final prediction was not generated by any individual model. Instead, the outputs of all recommendation methods were combined using a linear regression model implemented in the Weka machine learning toolkit. This ensemble strategy leveraged the strengths of each individual model while reducing their weaknesses.

The resulting system achieved an RMSE of approximately
```math
\boxed{0.256}
```
which was sufficient to obtain first place in the competition.

# Relationship with Probabilistic Matrix Factorization
The SVD component of the winning solution can be interpreted as a particular case of 'low-rank matrix factorization', which later became the foundation for methods such as 'Probabilistic Matrix Factorization (PMF)' and 'Non-negative Matrix'

## Conclusion
The KDD Cup 2007 winning solution demonstrated that combining multiple recommendation strategies yields substantially better results than relying on a single model.

Among these methods, low-rank matrix factorization emerged as the most influential idea and became the conceptual basis for modern recommender systems, including PMF, Bayesian PMF, SVD++, and neural collaborative filtering models used in contemporary platforms such as Netflix, Spotify, and YouTube.
